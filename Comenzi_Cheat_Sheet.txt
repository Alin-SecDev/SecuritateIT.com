Comenzi GitHub

Initial commit
Open git bash inside the folder you want to commit
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/Alin-SecDev/name_of_the_project.git
git push -u origin main

update the GitHub repo:
git add .
git commit -m "your comment about the update on repo"
git push

At the end of each day must push to the repo all the work that I have done. Make this into a habit. 
Be very careful when adding the commit message to be concise and specific about the update and if possible commit after the modification or updating a single file. This is how you keep track of the work that you done.
